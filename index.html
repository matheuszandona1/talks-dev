Pelo Começo!

<h1 class=" titulo ">
  Isso é um HTML qualquer
</h1>

Simples, puro, sem muito frufru por enquanto!
Uma tag sem estilização em HTML é uma marcação que define um elemento de conteúdo sem atribuir nenhum estilo ou formatação a ele. Geralmente, essas tags são usadas para estruturar o conteúdo da página da web.

O básico do CSS

<div class="fade">
  I’m slow and smooth
</div>

CSS é uma linguagem de estilo para definir a aparência de documentos web, permitindo a definição de estilos e layouts para elementos. Ele trabalha com HTML para criar páginas web visualmente atraentes e interativas.

O básico do Javascript

var titulo = document.getElementById("color-h1");
titulo.addEventListener("click", function() { titulo.classList.toggle("active"); });

JavaScript é uma linguagem de programação de alto nível que é usada principalmente para criar interatividade em páginas web. Ele permite a criação de funcionalidades dinâmicas, validação de formulários e manipulação de eventos do usuário, tornando as páginas web mais responsivas e interativas.

Mais JavaScript

 

Nesse exemplo fazemos uso de uma ferramenta muito utilizada no desenvolvimento de um site, uma API. essa API possibilita a reescrita de frases pré estabelecidas no JavaScript.
Uma API basicamente é um conjunto de informações ou de códigos para obter informações ou pra simplesmente não ser necessário criar.
O uso de API para escrita de código é como se fosse um atalho, ao invés de escrever várias linhas de código, a API ajuda com 3, 4 palavras e torna totalmente funcional.
